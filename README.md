# App de perros 🐶

Aplicación web que permite buscar fotos de perros según su raza, utilizando la [Dog API](https://dog.ceo/dog-api/).

## Tecnologías
- HTML
- CSS
- JavaScript (ES6)

## Cómo usar
1. Ingresá una raza en inglés en el campo de búsqueda
2. Hacé clic en **Buscar**
3. Se mostrarán las fotos de esa raza
